# Diabetes-prediction-project
Diabetes Prediction using Machine Learning
📌 Project Description

This project aims to predict whether a person is diabetic or not using machine learning techniques. The model is trained on medical attributes such as glucose level, BMI, age, blood pressure, insulin, and genetic factors.

📊 Dataset
Dataset used: Pima Indians Diabetes Datase
t
Features:
Pregnancies
Glucose
Blood Pressure
Skin Thickness
Insulin
BMI
Diabetes Pedigree Function
Age

Target:
Outcome (0 = Non-Diabetic, 1 = Diabetic)

⚙️ Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib / Seaborn
Streamlit (for deployment)
🤖 Machine Learning Models Used
Logistic Regression 
K-Nearest Neighbors (KNN)
Support Vector Machine (SVM)
Decision Tree
Random Forest(Best fit)
Gradient Boosting


Evaluation Metrics:
Accuracy
Precision
Recall
F1 Score
RMSE
Confusion Matrix
